# DR4-TP2-2026  
### TP2 — Desenvolvimento Web com .NET e Bases de Dados
Este projeto reúne exercícios práticos desenvolvidos em ASP.NET Core Razor Pages do TP2.

---

## 📋 Exercícios incluídos

1. **Razor Page com PageModel e Propriedade Vinculada**
2. **Model Binding via Handler Parameters**
3. **Validação Server-side com Data Annotations**
4. **Validação Client-side com Tag Helpers**
5. **Encapsulando Entrada em Objeto Complexo**
6. **Validação de Código ISO do País**
7. **Trabalhando com Vários Registros Simultaneamente**
8. **Roteamento com Parâmetros na URL**
9. **Criando URLs com Tag Helpers**
10. **Manipulação de Arquivos: Escrita**
11. **Manipulação de Arquivos: Leitura**
12. **Validação Customizada com ModelState**

---

## ▶️ Como executar o projeto

1. **Abra o projeto no Visual Studio**
2. Certifique-se de que o projeto está configurado para **ASP.NET Core com Razor Pages**
3. Pressione `Ctrl + F5` ou clique em **Iniciar sem depuração**
4. A aplicação abrirá no navegador, permitindo navegar entre as páginas dos exercícios

---

## 🛠️ Ambiente de desenvolvimento

- **Target Framework:** .NET 8
- **Versão da linguagem C#:** 12.0
- **IDE recomendada:** Visual Studio 2022 ou superior
